# gh0stshell
Shell Multiplexer


<img src="https://wslab.de/wslab/logo_gs_wh.png" height="50%"> 


## What is Gh0stshell?
Gh0stshell is a Shell Multiplexer. You can run a multihandler (listener) and connect many clients.

## Features
+ Manage multiply shells
+ Reverse Shells
+ HTTPS Encryption 

## Requirements
- Linux or BSD
- Python 2.7
- Socat 
- TmuxT

Tested on Debian
